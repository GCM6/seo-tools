# S5 · 冷启动与外链

> 来源：docs/seo.md（外链完全指南）、docs/seo1.md（自然流量思路）、docs/seo06.md §24（新站阶段优化）、§4（满月数据）、§15（外链节奏）、§25（外链因素）。

## 一、新站冷启动
- **第一指标是展示量，不是点击量**：上线一周展示量破百 = 内容合格信号；长期破不了百，先别急着上外链。
- **博客优先破局**：新站对商业落地页信任度低，博客（利他信息型）更易快速收录出排名；优先信息意图长尾词、问题词、对比类、top 榜单。
- **隐藏流量红利**：很多长尾词工具显示零搜索量，实际能跑出排名/点击，还能触发精选片段、AI Overview。
- **标准配置**：上线初期 20-30 个聚合落地页 + 20-30 篇基础博客，形成初步架构与内链即可；不要一次上太多页（防内容农场判定）。
- **节奏**：先靠博客拿基础流量与用户信号，聚合页通常 3-4 个月才发力，稳扎稳打。
- **B 端词可零外链出排名**：靠高质量首页/落地页，预算有限的新站也能冷启动（来源 seo1）。

## 二、外链的定位（来源 docs/seo.md）
- 外链对排名**非常有用**，至少占 SEO 总成本一半；持续投入能拉动关键词数与流量，停投则内容再更新也难突破。
- 但外链资源不稳定、受算法影响大，是高成本、有波动的环节；docs 作者经验认为其重要性远低于内容（原文"20-30% vs 70-80%"为经验估计，谷歌未公布任何权重比例，对外输出禁用具体百分比）〔官方勘误 → references/10-official-corrections.md C-3〕。

## 三、外链启动时机与标准流程〔官方勘误 → references/10-official-corrections.md C-2：给方案必须按合规档位披露〕
新站上线即可布局，无需等关键词出排名。docs 原三步（由自然到高权重）及其**官方合规定性**：
1. **社交分享铺垫**【官方允许】：先在 Facebook/领英分享内容，积累社交信号，模拟自然传播。
2. **基础低权重外链**【违规·不建议】：docs 原做法为上 Web 2.0、博客目录类「垃圾外链」以稀释锚文本、降低作弊嫌疑——主动批量制造低质链接属谷歌官方定义的 link scheme，且"降低作弊嫌疑"本身即规避检测，风险最高。合规替代：靠内容类型多样化（工具页/数据页/指南）自然获得多样外链。
3. **高权重外链**【灰色·高风险】：付费 Guest Post 若以传权重为目的且不加 `rel="sponsored"`，官方定性为 link spam（买卖链接）；实践普遍但属知情风险操作，**给方案时必须向用户披露官方定性与 SpamBrain 处罚可能**。合规替代：数字 PR、原创数据/研究换取自然引用、加 `sponsored` 的品牌曝光型投稿。

## 四、量级参考（来源 docs/seo.md / §10）【经验级行业观察，不作为建议输出；涉付费链接的合规风险见上节】
- 成熟站（1 年）：年新增 80-100+ 条。
- 成长站（半年）：约 20 条（基础 + 1 轮客座）即可支撑日点击 100+。
- 核心大词案例：造纸机核心词稳定首页用 7-8 个月，期间投 120-150 条 Guest Post（单条约 500 元）。

## 五、外链安全（核心：可接受小波动，绝不能被彻底惩罚）
- **结构多元混合**：nofollow + dofollow 搭配，高权重 + 低权重混合；新站初期不能全是高权重外链。
- **锚文本避免过度优化**：几乎不用精准关键词锚文本（风险极高，尤其多站点）。
- **节奏自然**：成熟期主动放缓，甚至数月不主动做外链，靠优质内容自然获得引用外链（每周自然新增 5-6 条更健康）。
- **算法波动认知**：算法更新致外链效果波动属正常；内容有价值 + 外链合规，算法稳定后通常恢复。最该规避的是「触底式惩罚」。

## 红线
- NEVER 新站零展示就大量上外链。
- NEVER 大量精准关键词锚文本 / 短时间批量上外链 / 工具自动跑外链。
- NEVER 全 dofollow 或全高权重外链。
- ALWAYS 建议付费外链/人工外链时，同步披露谷歌官方 link spam 定性与合规替代（见 references/10-official-corrections.md C-2 三档）。

## 自检
- [ ] 新站先看展示量破百再考虑外链。
- [ ] 外链按「社交→低权重→Guest Post」顺序，结构混合。
- [ ] 锚文本不过度优化，增长节奏自然。
- [ ] 外链投入与站点阶段/词难度匹配。
