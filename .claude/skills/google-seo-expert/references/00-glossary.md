# 术语速查表

> 单一真源：本 skill 所有术语在此定义，其他文件只引用不重定义。

| 术语 | 解释 | 关联站 |
| :--- | :--- | :--- |
| **聚合页（蜘蛛页）** | 整合全站相关内容、围绕核心商业意图的解决方案落地页，是网站权重基石，承担转化核心。短期不见效，通常 2-5 个月后排名超越博客。 | S2/S4 |
| **博客页** | 利他/信息型内容，覆盖长尾词，是新站冷启动主力，占外贸站 60-70% 自然流量；通过内链向聚合页传递权重。 | S4 |
| **双擎战术** | 同一核心主题同时布局「商业转化页（聚合页）+ 信息指南页（博客）」，两页 TDK/角度完全区分，互不蚕食，争取占据搜索结果两个位置，后期可按需 301/canonical 合并。 | S2/S4 |
| **关键词蚕食（cannibalization）** | 同一关键词站内多页同时参与排名，权重分散，导致排名不稳/互相打架。处理：内容合并 + canonical 或 301。 | S7 |
| **TDK** | Title（标题）/ Description（描述）/ Keywords，页面基础排名信号，自然融入目标词。 | S3 |
| **语义相关词（行业俗称"LSI 关键词"）** | 相关关键词，由搜索下拉、相关搜索拓展，丰富语义覆盖、避免核心词堆砌。〔术语勘误：谷歌官方（John Mueller）明确否认存在"LSI keywords"机制，"LSI"是行业俗称非谷歌技术，实践内核不变 → references/10-official-corrections.md C-5〕 | S4 |
| **NLP 语义优化** | 在语义相关词（LSI）覆盖之上融入行业专业术语、实体词、上下文关联词，让内容更专业、更贴合谷歌语义判定，利于被 AI Overview 引用。 | S4 |
| **E-E-A-T** | Experience（经验）、Expertise（专业）、Authoritativeness（权威）、Trustworthiness（可信）。强化方向：第一人称实操经验 + 权威数据溯源 + 资质案例。 | S4 |
| **KD（关键词难度）** | 第三方工具估算的排名难度，仅供参考；实际难度高度依赖域名权重。新站先做低 KD 词养权重。 | S2 |
| **黄金得分（Golden Score）** | WriterZen 等工具衡量词难度的指标，越低越易排名（30 分以内为低难度）。无 API。 | S2 |
| **DA/DR** | 第三方工具（Moz/Ahrefs）的域名权重评分，非谷歌官方指标，可被刷高，参考有限；降权后 DR 仍可能很高。 | S6 |
| **抓取预算（crawl budget）** | 谷歌对每个站点的抓取频次/总量上限，与权重正相关。无效页/小语种/标签页/404 会浪费它，导致新页不收录、老页掉排名。 | S7 |
| **PAA（People Also Ask）** | 搜索结果中的「其他人也在问」，反推用户真实问题，补充内容维度。 | S2/S4 |
| **Guest Post（客座投稿）** | 高权重外链类型，在他站发布带链接的优质文章。重质不重量。 | S5 |
| **Web 2.0 / 博客目录外链** | 低权重「基础外链」，作用不是传权重，而是稀释锚文本、丰富外链类型，降低作弊嫌疑。 | S5 |
| **nofollow / dofollow** | 链接属性。外链需两者混合分布，不能全是 dofollow。 | S5 |
| **canonical（规范标签）** | 告诉谷歌哪个是权威主页面，原页可访问、排名信号集中到主页。跨域名无效。 | S7 |
| **301 重定向** | 永久跳转，权重几乎完全传给目标页，原页废弃。用于彻底合并/改版。 | S7 |
| **AI Overview（AI 概览）** | 谷歌搜索顶部的 AI 总结。吃掉信息/定义类流量，但高质量专业内容可被引用获得精准曝光与询盘。 | S4/S6 |
| **GSC（谷歌站长工具 / Search Console）** | 谷歌官方数据，最准最及时（展示/点击/平均排名/出词）。判断效果以它为准。 | S6 |
| **首页/落地页/产品页/分类页** | 页面类型，按搜索意图与层级（L1 一级/L2 二级）规划，遵循一词一页一意图。 | S2 |
| **Vibe Coding** | 用 AI（Claude Code/Codex）以自然语言驱动完成 SEO 全流程（选词/内容/改图/技术修复）的工作方式。 | S9 |
| **证据等级标注** | 本 skill 主张的三级标注：【官方】谷歌/微软官方文档或公开声明；【共识】多来源独立研究一致；【经验】docs 作者单一来源实践估计（不得以事实口径对外输出）。定义与勘误条目见 `references/10-official-corrections.md`。 | 横切 |
